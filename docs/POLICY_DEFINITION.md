# 목적

공공기관의 정부권장정책(공공구매) 실적을 자동으로 계산하기 위한 정책 정의 문서이다.

이 문서는 시스템이 지원하는 정책과 계산 기준을 정의하며, 데이터 수집기(Connector)와 계산 엔진(Calculation Engine)의 기준 문서로 사용한다.

# 공통 계산 원칙

모든 정책은 아래 기준을 따른다.

- 사업자등록번호를 기준으로 기업을 식별한다.
- 인증 유효기간 내의 구매만 인정한다.
- 구매일을 기준으로 인증 여부를 판단한다.
- 동일 기업이 여러 정책을 동시에 보유할 수 있다.
- 하나의 구매건은 여러 정책 실적에 동시에 반영될 수 있다.
