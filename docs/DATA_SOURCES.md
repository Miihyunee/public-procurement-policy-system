# Purpose

본 문서는 Public Procurement Policy System에서 사용하는 모든 데이터의 출처(Data Source)를 정의한다.

각 데이터의 제공기관, 수집 방법, 갱신 주기 및 활용 목적을 관리하며, 시스템에서 사용하는 데이터의 신뢰성과 최신성을 확보하는 것을 목표로 한다.

# Collection Principles

- 공식 기관에서 제공하는 데이터만 사용한다.
- 사업자등록번호를 기준으로 기업 데이터를 연결한다.
- 데이터의 최신 여부를 확인할 수 있어야 한다.
- 가능한 경우 자동 수집을 우선한다.
- 모든 데이터셋은 수집 이력을 관리한다.
