# Purpose

본 문서는 Public Procurement Policy System에서 사용하는 모든 데이터의 출처를 정의한다.

각 데이터는 공식 기관에서 제공하는 자료를 기준으로 수집하며, 데이터의 갱신 주기와 수집 방법을 함께 관리한다.

# Collection Principles

- 공식 기관에서 제공하는 데이터만 사용한다.
- 사업자등록번호를 기준으로 데이터를 연결한다.
- 데이터의 최신 여부를 확인할 수 있어야 한다.
- 가능한 자동 수집을 우선한다.
- 수집 이력을 관리한다.
