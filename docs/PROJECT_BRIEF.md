# Public Procurement Policy System

## 프로젝트 목적

공공기관 구매담당자가 정부의 우선구매 정책 달성률을 쉽고 정확하게 관리할 수 있도록 지원하는 시스템을 개발한다.

---

## 프로젝트 목표

구매실적(Excel)을 업로드하면 자동으로

- 중소기업
- 여성기업
- 장애인기업
- 창업기업
- 사회적기업
- 녹색제품
- 중증장애인생산품
- 국가유공자 자활용사촌 생산품

등의 정책별 실적을 계산한다.

---

## 핵심 기능

### 1. 기업 및 인증정보 자동 수집

정부기관에서 공개하는 데이터를 자동으로 수집한다.

### 2. 구매실적 업로드

Excel 파일을 업로드한다.

### 3. 자동 매칭

사업자등록번호를 기준으로 기업을 찾는다.

### 4. 정책별 실적 계산

우선구매 실적을 자동 계산한다.

### 5. 대시보드

기관의 달성률을 시각적으로 표시한다.

---

## 개발 원칙

- Python 사용
- SQLite 사용
- GitHub 기반 개발
- GitHub Actions 자동 실행
- 가능한 모든 데이터는 자동 수집
- 데이터 변경 이력 관리
- 오픈소스 프로젝트로 개발

---

## 최종 목표

공공기관 구매담당자가 엑셀 계산 없이
우선구매 실적을 자동으로 관리할 수 있는 플랫폼을 구축한다.
